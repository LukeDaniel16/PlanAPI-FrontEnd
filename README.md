    * Vinícius Vieira.
* Frameworks utilizados:
  * Front-end:
    * Vue.js
    * Buefy
    * Nuxt.JS
  * Back-end:
    * ASP.NET Core     
    * ASP.NET WebAPI

## Como rodar o projeto - Após clonar:

```bash
# install dependencies
$ npm install

# Rodar em server localHost 3000
$ npm run dev

# Rodar para produção
$ npm run build
$ npm run start

```
